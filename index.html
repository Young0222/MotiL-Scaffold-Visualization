Scaffold Visualization Demo

A static front-end page for the MotiL scaffold visualization tool. Users can prepare their own molecular dataset CSV and pretrained checkpoint, set a few plotting parameters, and copy the exact local command needed to generate a scaffold-colored t-SNE figure and coordinate table.

Example output generated from the current BACE demo setup. The left panel shows scaffold structures, and the right panel shows scaffold-colored t-SNE points.

📥 Required Inputs

1. Molecular dataset CSV

  • The file must be a .csv.
  • The first column header should be smiles.
  • Each row should contain one molecule.
  • Extra columns are allowed. A common second column is a label such as Class.

2. Pretrained checkpoint

  • The file should be a PyTorch checkpoint, for example .pkl or .pt.
  • It should be compatible with torch.load(..., map_location="cpu").
  • It should contain a MotiL encoder state dict with keys such as encoder.W_i_atom.weight.
